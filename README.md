# LM Stick (Windows)

A portable, USB-friendly local LLM setup using Ollama. <br>
[Download](https://pub-bfb26aaa51024d16b54830768efdd3af.r2.dev/LM_Stick.zip)

## What it does
- Runs Ollama + a local model from a folder (USB drive friendly)
- Stores models in `.ollama/models` (portable)

## How to use
1. Plug in the USB / folder
2. Double-click run.exe
3. Use the Ollama GUI
4. Close the server terminal to end the program completely

## Notes
- Windows-only (for now)
- Do not unplug the USB while a model is running
